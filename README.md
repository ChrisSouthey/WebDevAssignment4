Dependencies:
bcrypt
connect-mongo
dotenv
express
Express-handlebars
express-session
mongo
mongodb
mongoose
passport
passport-local
nodemon

Install script:
npm install connect-mongo dotenv express express-handlebars express-session mongo mongodb mongoose passport passport-local bcrypt --save //Installs all the necessary dependancies
npm install nodemon --save-dev //Installs nodemon as a developer dependancy

How to run
type "npm run dev"
